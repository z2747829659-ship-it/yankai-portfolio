# Kevin / Yankai Zhao Portfolio

A dark, cinematic personal portfolio built with React, Vite, Framer Motion and Lucide icons.

## Run locally

```bash
npm install
npm run dev
```

## Replace the hero video

Put your video at:

```text
public/videos/kevin-hero.mp4
```

The page already includes the video tag and keeps a visual placeholder behind it.

## Edit content

Most text, project cards and strength cards live in:

```text
src/data/content.js
```
